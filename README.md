## ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/your-username/A402Pi.git

# Navigate to project folder
cd A402Pi

# Install dependencies
npm install
```

---

## ▶️ Usage

```bash
# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
